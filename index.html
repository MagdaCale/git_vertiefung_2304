Git Vertiefung

  1. Erstelle ein Projekt & eine Repo
  2. Lade das Team ein
  3. Wir erstellun uns eine Branch
    • git branch feature -> legt eine neue branch an
    • git checkout feature -> wechselt zur branch feature
    • git branch -b feature2 -> legt eine neue Branch an und wechselt direkt dorthin
    • git branch -> zeigt alle vorhanden branches an (* -> zeigt an, wo man sich gerade befindet)
    • git branch -d feature2 -> löscht den branch feature2
    • git branch -D feature2 -> Branch wird unabhängig von seinem Status und ohne Warnung gelöscht
